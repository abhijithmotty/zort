\# Changelog



All notable changes to ZORT will be documented in this file.



\## \[2.0.0] - 2025-10-11



\### Added

\- Complete Python rewrite for better stability

\- Async HTTP requests with aiohttp for 10x performance

\- Cross-platform support (Windows, Linux, macOS)

\- Real-time progress display

\- Better error handling and logging

\- Custom output directory option



\### Features

\- Smart pattern detection (tokens, APIs, parameters)

\- Vulnerability pattern matching (SQLi, XSS, LFI, RFI, SSRF, IDOR)

\- HTTP status intelligence (200, 401, 403, 405, 500+)

\- Concurrent processing with async/await

\- Comprehensive reporting system
